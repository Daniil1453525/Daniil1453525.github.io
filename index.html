Про мене

Професійний Python розробник з фокусом на штучний інтелект та автоматизацію. Маю глибокі знання у створенні та впровадженні AI-рішень для автоматизації процесів, обробки даних, розробки чат-ботів і генеративних моделей контенту. Володію широким спектром технологій для розробки веб-додатків, включаючи як бекенд-, так і фронтенд-технології. Постійно вдосконалюю навички та освоюю новітні інструменти для надання максимальної цінності у своїх проектах.

Навички

Python: Django, Flask, FastAPI, Pandas, NumPy, PyTorch, OpenCV
Штучний інтелект: TensorFlow, Scikit-Learn, Keras
Генерація контенту: GPT-3, ChatGPT, NLTK, SpaCy, Transformers
Автоматизація: Selenium, Scrapy, BeautifulSoup
Обробка даних: Jupyter Notebook, SQL
Хмарні сервіси: AWS, Google Cloud, Azure
